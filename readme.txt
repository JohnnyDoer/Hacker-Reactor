We are building a website where enthusiastic Cyber Security people can hone their skills by solving challenging labs. The labs would be contributed by verified individuals, the moderators. We would have a forum where people can have AMAs and can fathom bigger opportunities for themselves.


How to run:
1. Install python
2. Run "pip install -p requirements.txt"
3. Run "python manage.py runserver"
4. Open "127.0.0.1:8000" in a browser.
5. Click on the login link.
6. Insert Credentials if any or click on SignUp Now.
